# 潮新闻访问量监控台

一个面向 `https://tidenews.com.cn` 的简单阅读量监控工具。输入新闻标题、新闻链接和刷新间隔范围后，可以把多条新闻加入列表；每条新闻都能单独定时读取阅读量、停止监控和手动刷新。

## 功能

- 输入新闻标题和潮新闻链接
- 以列表形式管理多条新闻
- 每条新闻可单独开始或停止定时阅读量监控
- 每条新闻可单独设置随机刷新间隔（默认 `1s` 到 `3s`，最小 `1s`）
- 每条新闻可单独设置监控总时长，默认 24 小时，最长 24 小时，超时自动停止
- 手动刷新单条新闻并显示当前阅读量
- 使用本地 `JSON` 文件存储数据，部署简单

## 技术方案

- 后端：`Node.js` + `Express`
- 前端：原生 `HTML` / `CSS` / `JavaScript`
- 存储：`data/db.json`
- 监控逻辑：按设置的随机间隔读取页面数据并更新阅读量
- 读取逻辑：解析页面中的 `__NEXT_DATA__`，读取 `read_count` / `read_total`

## 快速开始

```bash
npm install
npm run dev
```

启动后打开 [http://localhost:3000](http://localhost:3000)。

## 示例链接

```text
https://tidenews.com.cn/news.html?id=3447701
```

## 接口说明

- `GET /api/news` 获取当前新闻列表
- `POST /api/news` 新增新闻
- `POST /api/news/:id/fetch` 手动读取一次阅读数
- `POST /api/news/:id/monitor` 开启或关闭定时阅读量监控，并更新秒级随机刷新范围和监控总时长
- `DELETE /api/news/:id` 删除新闻和历史
